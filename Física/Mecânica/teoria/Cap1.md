# Capítulo 1 — Cinemática: O Estudo do Movimento

## O que é Cinemática?

A **Cinemática** é o estudo do movimento dos corpos **sem se preocupar com as causas** desse movimento. Ou seja, analisamos *como* o corpo se move, não *por que* ele se move.

---

## Conceitos Fundamentais

### 📍 Posição
A posição de um corpo indica **onde ele está** em relação a um ponto de referência (origem).

- É representada pela letra **x** (ou **s**)
- Medida em **metros (m)**

### 📏 Deslocamento
O deslocamento é a **variação de posição** de um corpo:

```
Δx = x_final - x_inicial
```

> ⚠️ Deslocamento é diferente de distância percorrida! O deslocamento considera apenas a posição inicial e final.

### ⏱️ Velocidade Média
A velocidade média indica **quão rápido** um corpo se desloca:

```
v_m = Δx / Δt
```

Onde:
- **v_m** = velocidade média (m/s)
- **Δx** = deslocamento (m)
- **Δt** = intervalo de tempo (s)

### 🚀 Aceleração Média
A aceleração indica a **variação da velocidade** ao longo do tempo:

```
a_m = Δv / Δt
```

---

## Movimento Retilíneo Uniforme (MRU)

No **MRU** a velocidade é **constante** e a aceleração é **zero**.

**Equação horária da posição:**
```
x = x₀ + v·t
```

**Gráfico v × t:** linha horizontal (velocidade constante)
**Gráfico x × t:** linha reta inclinada

### Exemplo Resolvido

> Um carro viaja a 20 m/s. Qual a posição após 5 segundos, partindo de x₀ = 10 m?

```
x = 10 + 20 × 5
x = 10 + 100
x = 110 m
```

---

## Movimento Retilíneo Uniformemente Variado (MRUV)

No **MRUV** a aceleração é **constante** e diferente de zero.

**Equação horária da posição:**
```
x = x₀ + v₀·t + (a·t²)/2
```

**Equação horária da velocidade:**
```
v = v₀ + a·t
```

**Equação de Torricelli** (sem o tempo):
```
v² = v₀² + 2·a·Δx
```

### Exemplo Resolvido

> Um carro parte do repouso com aceleração de 3 m/s². Qual a velocidade após 4 s?

```
v = 0 + 3 × 4
v = 12 m/s
```

---

## Resumo

| Tipo | Velocidade | Aceleração |
|------|-----------|------------|
| MRU  | Constante | Zero |
| MRUV | Variável  | Constante |
